# Guzo API Module

## Getting started

```bash
# 1. Clone the repository.
git clone https://github.com/redeattsegaye/guzo-api.git guzo-api

# 2. Enter your newly-cloned folder.
cd guzo-api

# 3. Install dependencies
npm install

# 4. intall nodemon (for live-reload, optional)
npm install -g nodemon

# 5. Run the project (after running a mongo instance on localhost:27017)

npm start

# or optionally (watch the code and restart the server)
npm run dev
```
